# Vogue Dorado

**Vogue Dorado** is a modern, responsive **E-commerce website** that showcases and sells luxury products online. It features a clean design, smooth navigation, and a user-friendly interface. Built with **HTML, CSS, and Font Awesome**, this project serves as a front-end prototype of an online luxury store.

## Features
- Fully responsive design for desktop, tablet, and mobile.  
- Sticky navigation bar for easy browsing.  
- Hero section with promotional messaging.  
- Featured products section with categories like Watches, Accessories, Fragrances, Jewelry, and Bags.  
- Contact page with a functional contact form and company information.  
- Information sections for **Shipping Policy, Return Policy, and Warranty**.  
- Modern, clean UI with **Font Awesome icons** and **Google Fonts**.
